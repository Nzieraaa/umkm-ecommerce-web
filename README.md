## âœ¨ Fitur Utama

Sistem ini dibagi menjadi dua modul utama:

### 1. Modul User (Website)

* **Katalog Produk:** Halaman toko (`shop`) dan detail produk (`details`).
* **Transaksi:** Keranjang (`cart`), *checkout*, dan konfirmasi pesanan.
* **Akun Pengguna:** Register, Login, Manajemen alamat, Riwayat pesanan (`account-orders`).
* **Dukungan:** *Wishlist*, Ulasan Produk (`account-review`), Halaman Tentang Kami, Kontak.

### 2. Modul Admin

* **Produk & Data:** Manajemen produk, kategori, merek (brand), dan slider promosi.
* **Manajemen Pesanan:** Melihat semua pesanan (`orders`), detail pesanan, dan pelacakan pesanan.
* **Promosi:** Pengelolaan Kupon/Diskon.
* **Pengguna:** Manajemen daftar pengguna yang terdaftar.
